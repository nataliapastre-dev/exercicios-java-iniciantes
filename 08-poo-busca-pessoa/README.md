# Exercício POO – Busca de Pessoa

Exercício desenvolvido em Java para praticar a busca de dados em uma lista utilizando Programação Orientada a Objetos.

## Funcionalidades
- Cadastrar pessoas
- Listar pessoas cadastradas
- Buscar pessoa pelo nome
- Menu interativo no console

## Conceitos aplicados
- ArrayList
- Encapsulamento
- Scanner
- Estrutura de repetição
- Comparação de Strings

## Resultado
O programa permite buscar uma pessoa pelo nome e exibir seus dados quando encontrada.
