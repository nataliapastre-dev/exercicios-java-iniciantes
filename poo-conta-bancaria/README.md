Ao executar o programa, o usuário informa o nome do titular e interage com um menu no console para realizar operações bancárias.

Exemplo de execução:

Digite o nome do titular: natalia

=== MENU ===
1 - Depositar
2 - Sacar
3 - Consultar saldo
0 - Sair
Escolha uma opção: 1
Digite o valor para depósito: 300
