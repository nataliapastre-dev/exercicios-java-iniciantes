# Exercício POO – Lista de Pessoas

Exercício desenvolvido em Java para praticar Programação Orientada a Objetos utilizando ArrayList.

## Funcionalidades
- Cadastrar pessoas
- Listar pessoas cadastradas
- Menu interativo no console

## Conceitos aplicados
- Classes e objetos
- Encapsulamento
- ArrayList
- Scanner
- Estrutura de repetição

## Resultado
O programa permite o cadastro de pessoas via console, exibindo mensagem de confirmação após o registro.
